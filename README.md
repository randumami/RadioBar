# RadioBar
Macos menubar app that plays DR radio (Danish Radio)

_version 2.0_ - is in App Store

Clean simple app to stream radio channels, easy add your own streams.

this version now show artist and title when you hover over the icon in the menubar. Though only for radio channels that embed this info in their streams :-) 

this app lives in your mac's menubar. currently it streams danish radio channels.
The app can be set to autoreload when you log in. You can also add or remove the pre defined channels
